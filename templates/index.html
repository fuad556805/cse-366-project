About nl2sql

nl2sql is a hybrid natural-language-to-SQL tool. It combines a machine-learning intent classifier with rule-based matching to turn plain-English questions into real SQL queries — no coding required.

How it works:

  • Upload any CSV file — the columns and sample values are read automatically.
  • Type a question like "average purchase amount by location".
  • The AI detects your intent (SELECT / COUNT / AVG / MAX / MIN / SUM), matches the right columns, and builds a safe SQL query.
  • The query runs against your data and results are shown instantly.

No question about your data ever leaves this app — everything runs locally.

Help & Tips

1. Upload a dataset

  • Drag & drop or click to browse a .csv file.
  • The first row must contain column headers.

2. Ask a question

  • show female patients older than 30
  • average purchase amount by location
  • top 5 highest salary
  • count students where district is Dhaka

3. Tips for best results

  1. Mention the column name in your question when possible (e.g. "purchase amount", "review score").
  2. Use words like average, total, maximum, minimum for aggregates.
  3. Use top N / lowest N for ranked results.
  4. Use by <column> to group results.

If a question is too ambiguous, the app will show a clear error instead of guessing the wrong column.
